KSYAQPlayer Class Reference

Inherits from NSObject
Declared in KSYAQPlayer.h

Overview

基于AudioQueue的音乐播放器

* 提供声音播放的功能
* 通过回调拉取新的数据
* 提供音量和音调调节功能
* 能将播放后的音频数据通过回调送出 (回调的数据为音效处理之后的数据)
