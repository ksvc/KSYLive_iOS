Overview

叠加logo的滤镜

  1. 输入的 UIImage 叠加到输入KSYGPULogoFilter的纹理上
  2. 可以指定UIImage在纹理上的位置和透明度
  3. 通过clearLogo, 可以清除纹理上的图片
