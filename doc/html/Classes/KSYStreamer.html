Overview

金山云直播推流SDK iOS版提供了iOS移动设备上的实时采集推流功能。

  • 音频编码采用AAC编码，码率可配置;
  • 视频频编码采用H.264编码，码率可配置;
  • 支持 RTMP 协议直播推流;
  • 支持前后摄像头切换;
  • 多分辨率编码支持;

Discussion

  • _videoDimension = KSYVideoDimension_Default;
